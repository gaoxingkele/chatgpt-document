# 中英混合语言规范

## 专有名词
- 首次出现：中文译名（英文原名），如"贾利斯科新世代卡特尔（CJNG）"
- 后续引用：使用缩写或中文简称
- 人名：中文音译·英文原名，如"奥塞格拉·塞万提斯（Nemesio Oseguera Cervantes）"
- 机构：中文全称（英文缩写），如"美国缉毒局（DEA）"

## 引述
- 关键声明保留英文原文 + 中文翻译
- 格式：美方声明称"non-state armed groups"（非国家武装团体）
- 长引述（>30 词）单独成段，用引用块 >

## 数据与来源
- 统一格式：据[来源名称]报道/数据
- 日期：YYYY年M月D日
- 货币：XXX亿美元
- 百分比：XX%

## 禁止
- 整段英文不翻译直接引用
- 中英文混用半句话（如"进行了counter cartel行动"→"进行了反卡特尔行动"）
- ISO 时间戳出现在正文
- 裸 URL 出现在正文
- HTML 实体（&amp; 等）
